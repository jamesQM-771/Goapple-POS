# Guía Práctica 2: Selección de Frameworks y Capas
## Selección de componentes
Un Framework PHP aporta una colección de librerías y estructuras que aceleran el desarrollo (MVC). **Laravel** ofrece un ecosistema pesado y robusto con su propio ORM (Eloquent), mientras que **Slim** es un microframework ideal para APIs rápidas de bajo acoplamiento.

## Criterio de Reuso y Decisión
Ayuda estableciendo una convención estructurada, reduciendo el código espagueti. **Decisión:** Para este proyecto utilizaremos **PHP Nativo**, ya que el objetivo pedagógico de la materia es construir la arquitectura desde cero para dominar el enrutamiento lógico sin depender de herramientas externas automatizadas.
